### 工具使用说明
* 下载下来后的步骤
* 1 - 安装 “CheatEngine74”
* 2 - 解压绿色免安装版的易语言开发工具 “e_5.93_Green.7z”
* 3 - 再解压 “内存读写支持库2.3.zip”，并根据里面的说明文件把其中的两个关键文件放到 “e_5.93_Green” 的 lib 和 static-lib 目录下
* 4 - 用易语言开发工具打开这里的“真单就心丸_轮回修仙路_数据工程.e”工程的时候，记得要引用这里的 “精易模块V7.6.5” 目录下的 “精易模块[v7.6.5].ec” 工具包
